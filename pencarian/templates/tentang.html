Tentang Aplikasi

Aplikasi pencarian judul tugas akhir ini dibangun menggunakan library tkinter dengan menggunakan Bahasa Pemrograman Python. Aplikasi ini terdiri dari 4 fitur antara lain Login, Lihat daftar judul TA, Pencarian judul, Tentang Aplikasi, Logout.

Fitur-Fitur:

  • Login: Pengguna diminta untuk melakukan login dengan memasukkan username dan password, Kemudian pengguna harus memasukkan file Json untuk dapat masuk ke aplikasi.
  • Lihat Daftar Judul TA: Pengguna dapat melihat daftar judul TA. pada aplikasi ini judul TA yang ditampilkan hanya lulusan tahun 2021 dan 2022.
  • Pencarian Judul: Pengguna dapat mencari daftar judul TA. Pada fitur ini pengguna dapat memilih hasil pencarian berdasarkan 4 data yaitu berdasarkan Nomor TA, Nama Penulis, NIM Penulis, dan Judul TA.
  • Logout: Pengguna dapat keluar dari aplikasi dengan menekan tombol Keluar pada portal Utama.
